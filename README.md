# Task CRUD API

A simple Task Management CRUD API built using FastAPI.

## Features

- Create Task
- Read All Tasks
- Read Task by ID
- Update Task
- Delete Task
- Health Check
- Swagger Documentation

## Tech Stack

- Python
- FastAPI
- Pydantic
- Uvicorn

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## API Documentation

Open:

```
http://127.0.0.1:8000/docs
```
